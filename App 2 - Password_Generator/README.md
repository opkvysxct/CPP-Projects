# Password Generator App written in C++

## This app was created to learn the use of

- Enums
- Structs
- std::vector
- std::map
- std::default_random_engine
- std::uniform_int_distribution

## Use

Follow commands in console.

## Possible options

- Length of password
- Use of big letters
- Use of small letters
- Use of numbers
- Use of special characters
